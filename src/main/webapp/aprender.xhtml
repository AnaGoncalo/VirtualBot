Comandos de Movimentação

  • frente(1)

    Movimenta o robô para frente, deve ser passado como paramêtro quantas unidades de tempo/espaço o robô deverá pecorrer.

  • direita(1)

    O robô faz uma curva para direita no seu eixo.

  • esquerda(1)

    O robô faz uma curva para esquerda no seu eixo.

  • parar()

    O robô para de se movimentar.

Sensores

  • ultra()

    Sensor ultrasom, está posicionado na frente do robô, funciona como os olhos do robô.

Estruturas condicionais

  • se(ultra()>1) entao {frente(1)}

    Condição "Se", o robô só irá realizar o bloco de comandos se a condição for verdadeira

Estruturas de repetição

  • repita 2 vezes{frente(1)}

    Comando repetir, o bloco de comandos irá ser repetido n vezes
